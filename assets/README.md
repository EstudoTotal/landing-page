# Estrutura de assets

## 📁 images/
Coloque aqui todas as imagens do site.

Exemplos de nomes sugeridos:
- `nazli-foto.jpg` → foto da mentora (seção "Conheça a Nazli")
- `hero-bg.jpg` → imagem de fundo do hero (opcional)
- `og-image.jpg` → imagem para compartilhamento em redes sociais (1200×630px)

Para referenciar no HTML:
```html
<img src="assets/images/nazli-foto.jpg" alt="Foto da Nazli" />
```

## 📁 videos/
Coloque aqui os vídeos do site.

Exemplos de nomes sugeridos:
- `apresentacao.mp4` → vídeo de apresentação do curso
- `depoimento-aluno.mp4` → depoimento em vídeo

Para referenciar no HTML:
```html
<video src="assets/videos/apresentacao.mp4" controls></video>
```

> Dica: prefira vídeos em formato .mp4 (H.264) para maior compatibilidade entre navegadores.
